# NexusDB

Check a mod on Nexus Mods without having to go to Nexus Mods!

**NexusDB** is a Nexus Mods mod tracker. Simply put in your key, game ID (domain) and mod ID, and you have the most info you need!

Your API key will be saved locally in your browser.
